Top 50 songs' features

The data fetched using spotify API consists of artists, songs and their features. Each one is described in a range from 0 to 1.

  • danceability - describes how suitable a song is for dancing based on a combination of musical elements. A value of 0 is least danceable and 1 is most danceable.
  • acousticness - measures whether the track is acoustic; 1.0 represents high confidence that it is
  • energy - measures intensity and activity of a track
  • instrumentalness - predicts whether a track contains no vocals
  • liveness - detects the presence of an audience. Higher value represents increased probability that the song was performed live
  • speechiness - detects the presence of spoken words. If a recording is more speech-like, its value is closer to 1.0
  • valence - describes positiveness conveyed by a track. Songs with high valence sound more positively

Bar chart below allows you to compare average score for each feature in your top 50 songs in short time and long time. Short time results are based on your streaming in the last month.
